## Package options

ArchaeoCal uses the following
[`options()`](https://rdrr.io/r/base/options.html) to configure
behaviour:

- `ArchaeoCal.verbose`: a
  [`logical`](https://rdrr.io/r/base/logical.html) scalar. Should R
  report extra information on progress? Defaults to
  [`interactive()`](https://rdrr.io/r/base/interactive.html).
